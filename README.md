## Setup
1. Clone repo
2. Server:
   - cd server
   - cp .env.example .env (edit)
   - npm install
   - npm run dev
3. Client:
   - cd client
   - cp .env.example .env
   - npm install
   - npm run dev

## API
List endpoints: GET /api/posts, GET /api/posts/:id, POST /api/posts, PUT /api/posts/:id, DELETE /api/posts/:id, GET/POST /api/categories, POST /api/auth/register, POST /api/auth/login
